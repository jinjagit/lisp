## shell
`$ sbcl` open shell  
`* (quit)` close shell  

## run
note bash path in first line (find correct path with `$ which sbcl`)  
grant file permissions. e.g. `$ chmod 0755 hello_world.lisp`   
run: `$ ./hello_world.lisp`
